## How to use

### Provide a description

You can provide a description to be used within the embedded PageBanner.

<Preview>
  <Story name="Description">
    <Section title="Design and content" href="#" description="Guidance and standards for designing Home Office services." navigation={[
      { href: '#', text: 'Design system' },
      { href: '/design-and-content/content-style-guide', text: 'Content style guide' },
      { href: '/design-and-content/professional-standards', text: 'Professional standards and guidance' }
    ]}>
      My content.
    </Section>
  </Story>
</Preview>

### Homepage

On a home page, you might want to not provide any navigation items.

<Preview>
  <Story name="Homepage">
    <Section title="Design and build services that work for everyone" href="#" description="Use the UCD Manual to create consistent services that put users first.">
      My content.
    </Section>
  </Story>
</Preview>
